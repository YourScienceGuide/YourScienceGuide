# YSG Frontend Rules
- Simplicity and educational impact are the highest priorities.
- UI must be completely free of clutter, extraneous sidebars, or placeholder ads.
- ALL links and navigation must open in the same tab (`target="_self"` or standard Next.js `<Link>`). NEVER use `target="_blank"`.
- Use Tailwind CSS and shadcn/ui for clean, minimalist styling.
- Emphasize whitespace and highly readable typography for students.
